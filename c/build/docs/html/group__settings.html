Detailed Description

  • Messages for reading and writing the Piksi's device settings. These are in the implementation-defined range (0x0000-0x00FF), and intended for internal-use only. Please see the accompanying description of settings configurations for more details. Note that some of these messages taking a request from a host and a response from the Piksi share the same message type ID.
